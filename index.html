# 📱 КАК ПОЛУЧИТЬ APK — ПОШАГОВАЯ ИНСТРУКЦИЯ ## Что у тебя есть в этой папке: ``` lx74z-app/ ├── www/ │ ├── index.html ← Твоё приложение │ ├── manifest.json ← Настройки PWA │ ├── icon-192.png ← Иконка приложения │ └── icon-512.png ← Большая иконка ├── .github/ │ └── workflows/ │ └── build-apk.yml ← Инструкция для GitHub: как собрать APK ├── package.json └── capacitor.config.ts ``` --- ## ШАГ 1: Создай аккаунт GitHub 1. Зайди на https://github.com 2. Нажми **Sign up** (вверху справа) 3. Введи email, пароль, имя пользователя 4. Подтверди email --- ## ШАГ 2: Создай новый репозиторий 1. После входа нажми **+** (вверху справа) → **New repository** 2. **Repository name**: `lx74z-app` 3. Выбери **Private** (чтобы код был только у тебя) 4. Нажми **Create repository** --- ## ШАГ 3: Загрузи файлы на GitHub ### Вариант А — через браузер (проще): 1. На странице репозитория нажми **uploading an existing file** 2. Перетащи ВСЕ файлы и папки из скачанного архива 3. ⚠️ ВАЖНО: папки `.github/workflows/` нужно создать вручную: - Нажми **Create new file** - В имени файла напиши: `.github/workflows/build-apk.yml` - Скопируй туда содержимое файла `build-apk.yml` - Нажми **Commit new file** ### Вариант Б — через GitHub Desktop (удобнее): 1. Скачай GitHub Desktop: https://desktop.github.com 2. Войди в свой аккаунт 3. File → Add local repository → выбери папку `lx74z-app` 4. Publish repository → выбери Private 5. Готово — все файлы загружены! --- ## ШАГ 4: Запусти сборку APK 1. Зайди в свой репозиторий на GitHub 2. Нажми вкладку **Actions** (вверху) 3. Слева найди **Build Lx74z APK** 4. Нажми **Run workflow** → **Run workflow** (зелёная кнопка) 5. Подожди 5-15 минут — GitHub сам собирает APK! --- ## ШАГ 5: Скачай готовый APK 1. Когда появится зелёная галочка ✅ — сборка готова 2. Нажми на название запуска 3. Внизу страницы найди раздел **Artifacts** 4. Нажми **Lx74z-debug-APK** → файл скачается --- ## ШАГ 6: Установи на телефон 1. Перенеси `app-debug.apk` на телефон 2. В настройках Android: **Безопасность** → включи **Разрешить установку из неизвестных источников** 3. Открой файл APK → **Установить** 4. Готово! 🎉 --- ## ⚠️ ВАЖНЫЕ МОМЕНТЫ: ### Про Play Market: - Debug APK нельзя загрузить в Play Market - Для Play Market нужен **подписанный (signed) APK** или **AAB** - Это требует платного аккаунта разработчика ($25 один раз) - Если нужно — напиши, добавлю инструкцию для подписи ### Если сборка упала с ошибкой: - Зайди в Actions → нажми на красный X - Раскрой упавший шаг и скопируй ошибку - Покажи мне — помогу исправить --- ## 🔄 Как обновить приложение в будущем: Просто измени `www/index.html` на GitHub → Actions автоматически пересоберёт APK --- *Если что-то непонятно — спроси, помогу разобраться! 💜*
